# app/server/static 模块

`app/server/static/` 保存 Web 面板的前端资源。

## 关键文件

- `index.html`：同步方案配置、任务操作和日志面板结构。
- `styles.css`：面板样式、响应式布局和危险模式提示样式。
- `app.js`：前端状态管理、配置保存、任务触发、引导聚焦和二次确认逻辑。

## 维护注意

- 修改同步模式下拉框时，需要同时检查后端 `server.py` 的 `media_mode` 支持范围。
- 高风险“下载后删除 iCloud 云端”必须保留明显提示和二次确认。
- 路径展示面向普通用户只显示照片、视频、备忘录最终保存位置；如果当前服务仍在使用旧路径，只用一句话提示停止并重新启动应用后生效。
- 前端请求统一通过相对路径访问 Flask API，避免写死 NAS IP 或端口。
