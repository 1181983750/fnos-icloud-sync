# app/ui 模块

`app/ui/` 保存 fnOS 桌面入口配置和备用启动页脚本。

## 关键文件

- `config`：fnOS 桌面入口声明，当前使用飞牛标准 `port/url` 代理方式把桌面入口直达 Web 面板。
- `index.cgi`：备用启动检测页脚本，保留给后续排查或需要恢复本地等待页时使用。
- `images/`：入口图标资源。

## 维护注意

- 桌面入口当前配置为 `showDesktop=true`、`allUsers=true`、`noDisplay=false`。
- `index.cgi` 需要保持可执行权限，安装脚本会尝试 `chmod +x`。
- 默认入口不要再从浏览器侧探测 `当前域名:8080`，否则外网域名没有暴露裸端口时会一直停在等待页。
- 如果将来重新启用启动页，文案应面向普通用户，不要在首屏展示端口、候选入口、Docker、依赖、`icloudpd` 等调试细节。
- 页面可以提示“停止并重新启动应用”等用户可执行动作；更深入的排查命令应放在日志或文档里。
